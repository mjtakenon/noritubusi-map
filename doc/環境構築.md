# 環境構築

## Requirements

- Docker

## Setup

```bash
$ docker-compose up -d
```

## 動作確認

### mondodb

```bash
# コンテナを起動(デーモンとして)
$ docker-compose up -d

# mongodb の CLI に入る
$ docker-compose exec mongodb mongo
...
>
```

### app (Flask)

ブラウザで `http://localhost:5050` にアクセスして、 **Hello, Flask** と表示されるか確認する
