# 乗り潰しマップ自分用メモ

* 鉄路乗り潰すオタク用ツール

## 仕様

* ブラウザで動く
* これらの情報を表示
  * 地図
  * 路線図
    * 乗り潰したところは色を変える
    * JRと私鉄でも変える
* 機能
  * 乗り潰し路線の登録/削除
  * 地図上で選択できるよう
  
* 詳細
  * 地図で駅を選択するとその駅から乗り換えなしで行ける路線の駅を色を変えて表示
  * 色が変わった駅の中から降車駅を選択し乗り潰し登録
